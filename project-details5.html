Roles & Responsibilities

Project manager and lead developer overseeing the design and implementation of the Smart Mirror.

Project Details

The Smart Mirror integrates smart home features and personal assistant capabilities. It provides users with weather updates, calendar notifications, and news headlines, all displayed on the mirror surface.

Impacts

This project enhances the daily routine by providing a convenient and interactive interface for accessing essential information. It integrates seamlessly with existing smart home devices, improving user experience and connectivity.
